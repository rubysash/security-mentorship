# Certifications (2026+)

Generally you get foundations, then specialize in specific vendors/products afterwards. Certs "expire" but are easy to renew.

## Entry Level (Start Here)
*   **AWS Cloud Practitioner** – Cloud fundamentals
*   **Azure Fundamentals** (AZ-900) – Microsoft cloud basics
*   **LPI Linux Essentials** – Linux basics
*   **CompTIA A+** – IT fundamentals, hardware, troubleshooting
*   **CompTIA Network+** – Networking concepts, protocols, infrastructure

## Security Foundations
*   **CompTIA Security+** – Industry standard entry security cert
*   **CompTIA Linux+** – Deeper Linux administration
*   **Cisco CCST Cybersecurity** – Entry-level Cisco path

## Intermediate & Specialization (Hot 2026)
*   **CCSP (Certified Cloud Security Professional)** – Cloud governance and architecture (High ROI in 2026)
*   **OSCP (Offensive Security Certified Professional)** – Gold standard for hands-on penetration testing
*   **CISSP (Certified Information Systems Security Professional)** – Management, operations, and risk
*   **CompTIA CySA+** – Threat detection with an emphasis on AI-driven SOC operations
*   **CompTIA PenTest+** – Modern offensive security basics
*   **AWS Security Specialty** – Cloud-native security and AI guardrails
*   **Azure Security Engineer** (AZ-500) – Microsoft cloud security
*   **Cisco CyberOps Associate** – Incident response and SOC foundations

## Vendor Specialist Paths
### AWS Path (The AI Factory)
*   **Solutions Architect Associate/Professional** – Focus on designing GPU clusters (P5 instances), Amazon Bedrock orchestration, and resilient AI infrastructure.
*   **Security Specialty** – Deep dive into securing the AI supply chain, model weights, and training data.

### Palo Alto Path (Precision AI)
*   **Network Security Professional** – Securing the edge with SASE (Prisma Access) and AI-driven SOC operations (Cortex XSIAM).
*   **Network Security Architect** – Designing Zero Trust architectures for autonomous AI agents.

### Cisco Path (Zero Trust & Automation)
*   **CCNP Security (SCAZT)** – Focus on Secure Cloud Access, Zero Trust, and eBPF-based workload protection for containers.
*   **CyberOps Professional** – Advanced incident response for AI-integrated applications and complex hybrid environments.
*   **Splunk Core Certified Consultant** – Mastery of the industry-standard SIEM for observability and automated SOC operations.

### NVIDIA Path (AI Hardware & Infrastructure)
*   **NVIDIA Associate (NCA-AIIO)** – Foundational knowledge of GPU architecture (Grace Hopper/Blackwell), NVLink, and InfiniBand networking.
*   **NVIDIA Professional (NCP-AII)** – Hands-on deployment of DGX/HGX clusters and BlueField DPU optimization.

### CrowdStrike Path (XDR & Threat Hunting)
*   **CrowdStrike Certified Falcon Professional (CCFP)** – Endpoint detection, response (EDR), and the "adversary-centric" approach to security.
*   **CrowdStrike Certified Search Specialist** – Mastery of Next-Gen SIEM and log-scale architecture for machine-speed hunting.

### Fortinet Path (Network-Security Convergence)
*   **NSE 7 / Fortinet Certified Professional (Network Security)** – Focus on Hybrid Mesh Firewalls, SD-WAN, and ASIC-powered performance.
*   **NSE 8 / Expert** – Advanced architecture involving SASE, ZTNA, and OT (Operational Technology) security.

### Zscaler Path (Zero Trust Exchange)
*   **Zscaler Certified Cloud Administrator (ZCCA)** – Managing secure internet access (ZIA) and private application access (ZPA) without a traditional VPN.
*   **Zscaler Certified Cloud Architect (ZCCA-IA)** – Designing "Direct-to-Cloud" architectures and SSE (Security Service Edge) for global enterprises.

### Okta Path (Identity as the Perimeter)
*   **Okta Certified Professional/Administrator** – Mastery of SSO, MFA, and Lifecycle Management.
*   **Okta Certified Consultant** – Designing advanced identity workflows, including securing non-human "AI Agents" and API identities.

## 2026 Trends & Compensation
*   **Platformization:** Enterprises are moving away from "best-of-breed" point products toward unified platforms (e.g., Palo Alto's Cortex, CrowdStrike's Falcon, Cisco's Security Cloud).
*   **Agentic AI Governance:** Managing and securing autonomous AI agents within the enterprise.
*   **Post-Quantum Readiness:** Migrating cryptographic systems to quantum-safe standards (Fortinet and Cisco lead here).
*   **AI-Driven SOCs:** Transitioning from manual log analysis to machine-speed automated response.
*   **Salary Outlook:** CCSP, OSCP, and CrowdStrike/Zscaler specialists command significant premiums in the 2026 market.

## Suggested Paths
*   **Defensive Analyst:** Network+ → Security+ → CySA+ → CCSP
*   **Offensive Engineer:** Network+ → Security+ → PenTest+ → OSCP
*   **Cloud Security Architect:** AWS Solutions Architect → AWS Security Specialty → CCSP
*   **AI Infrastructure Specialist:** NVIDIA Associate → AWS Solutions Architect → NVIDIA Professional
